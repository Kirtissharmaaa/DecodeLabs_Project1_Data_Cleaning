# DecodeLabs Project 1 - Data Cleaning

## Objective
The objective of this project was to clean and prepare a raw e-commerce dataset by handling missing values, duplicates, and incorrect data.

## Dataset Overview
- 1200 rows
- 14 columns
- E-commerce order data

## Tasks Performed
- Identified missing values in the Coupon Code column
- Replaced 309 missing values with "No Coupon"
- Checked for duplicate records (0 duplicates found)
- Verified data types and formats
- Validated Total Price calculations
- Investigated floating-point precision differences

## Tools Used
- Python
- Pandas
- Google Colab

## Results
The dataset was successfully cleaned and prepared for further analysis. All missing values were handled, data types were verified, and no duplicate records were found.
